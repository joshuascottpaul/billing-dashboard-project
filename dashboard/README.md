## Open the Dashboard

### Option 1: Direct File Access

```bash
# From project root
open dashboard/index.html
```

**Note:** Some browsers block CSV loading via `file://`. If you see a "Data load error" message, use Option 2.

### Option 2: Local Server (Recommended)

```bash
# Start server on port 8000
python3 -m http.server 8000

# Open in browser
open http://localhost:8000/dashboard/
```

### Option 3: Homebrew CLI

```bash
# If installed via Homebrew
billing-dashboard serve

# Or custom port
billing-dashboard serve --port 3000
```

## First Time Setup

### Step 1: Run Analytics Pipeline

The dashboard reads from CSV files generated by the analytics pipeline:

```bash
# Use sample data (included in repo)
cp nc-2002-2026-sample.xlsx nc-2002-2026.xlsx
bash analysis/run.sh
```

### Step 2: Open Dashboard

```bash
open dashboard/index.html
# or
open http://localhost:8000/dashboard/
```

### Step 3: Verify Data Loaded

You should see:
- ✅ 6 KPI cards with values
- ✅ Charts rendered
- ✅ Tables populated
- ✅ Data freshness badge in header

If you see **"Data load error"**, check the [Troubleshooting](#troubleshooting) section.

## Global Filters

One filter bar controls the entire dashboard:

| Filter | Description | Example |
|--------|-------------|---------|
| **Year from** | Start of time window | `2024` |
| **Year to** | End of time window | `2025` |
| **Focus year** | KPI year + YoY comparison base | `2025` |
| **Customer** | Filter all tables by customer name | `Acme Corporation` |
| **Top N** | Number of customers in Top-N charts/tables | `10` |
| **Include forecast** | Show/hide forecast on monthly chart | ✓ / ✗ |
| **Compare vs prior year** | Add YoY comparison to KPIs | ✓ / ✗ |

### Saved Views

Quick presets for common analysis scenarios:

| View | Settings | Use Case |
|------|----------|----------|
| **Default overview** | 2024-2025, Focus 2025, Top 10 | General review |
| **Collections review** | Latest year, Top 20, Collections preset | A/R analysis |
| **Growth review** | 3-year window, Compare mode | Trend analysis |
| **Concentration risk** | Focus year, Top 5 | Customer dependency |

Apply a view with the **Apply view** button.

### Active Filter Chips

The filter chip row shows your current settings at a glance:

```
[2024 → 2025] [Focus: 2025] [Top 10] [Forecast: On] [Compare: Off] [Default Overview]
```

## KPI Cards

6 key metrics with risk indicators:

| KPI | Description | Risk Thresholds |
|-----|-------------|-----------------|
| **Invoices Total** | Total invoice amount (focus year) | — |
| **Payments Total** | Total payments collected (focus year) | — |
| **Net Total** | Invoices + Credits - Payments | 🔴 Negative |
| **Top-1 Share** | Largest customer concentration | 🟡 >30%, 🔴 >50% |
| **Latest DSO** | Days sales outstanding (3-month avg) | 🟡 >30, 🔴 >60 |
| **High-Risk A/R** | Customers with >50% in 90+ bucket | Count shown |

## Charts

### Export Options

Each chart has export buttons in the header:

- **Chart CSV** - Download chart data as CSV
- **PNG** - Download chart as PNG image

### Chart Types

| Chart | Type | Scope | Description |
|-------|------|-------|-------------|
| **Top-N Cross-Tab** | Grouped bars | Focus | Focus year vs prior year comparison |
| **Yearly Summary** | Grouped bars | Date range | Annual invoices/payments/credits |
| **Monthly + Forecast** | Line | Date range | Monthly trend with 12-month forecast |
| **Concentration** | Multi-line | Date range | Top 1/5/10 cumulative share |
| **Invoice Drift** | Multi-line | Date range | Median/P75/P90/P95 invoice amounts |
| **Retention Cohorts** | Stacked bars | Date range | New/retained/reactivated/churned |
| **Size Mix** | 100% stacked bars | Date range | Revenue by invoice size bucket |
| **Currency Exposure** | Donut | Focus year | Revenue by currency |
| **Country Exposure** | Donut | Focus year | Revenue by country |

## Tables

### Features

- **Sortable:** Click any column header to sort
- **Resizable:** Drag the grip on the right edge of headers
- **Export:** Click "Export CSV" to download table data
- **Customer links:** Click customer names to open drill-down drawer

### Table Columns

| Table | Key Columns |
|-------|-------------|
| **Top Customers** | Rank, Customer, Invoice Total, Share, Cumulative |
| **YoY Delta** | Customer, Prior Year, Focus Year, Delta, Delta % |
| **Collections Risk** | Customer, Open Balance, Buckets (0-30, 31-60, 61-90, 90+), Share 90+ |
| **Payment Behavior** | Customer, Matched Amount, Median Days to Pay, Late Share >60d |
| **Unmatched Invoices** | Customer, Debit Date, Unmatched Amount, Age Days |
| **Recent Anomalies** | Type, Date, Customer, Amount, Threshold, Note |

### Customer Drill-Down

Click any customer name to open a side drawer with:

- Focus year sales
- YoY change
- Open balance
- Late share (>60d)
- Customer sales by year chart
- Recent anomalies for that customer

## URL State (Bookmarkable)

Dashboard state is encoded in URL parameters:

```
http://localhost:8000/dashboard/?yf=2024&yt=2025&fy=2025&n=10&fc=0&cm=0
```

| Param | Description | Example |
|-------|-------------|---------|
| `yf` | Year from | `2024` |
| `yt` | Year to | `2025` |
| `fy` | Focus year | `2025` |
| `n` | Top N | `10` |
| `fc` | Include forecast (0/1) | `0` |
| `cm` | Compare mode (0/1) | `0` |

**Tip:** Bookmark URLs with specific filter states for quick access.

## Troubleshooting

### "Data load error: Failed to fetch"

**Cause:** Browser security blocks `file://` CSV loading.

**Solution:** Use a local server:

```bash
python3 -m http.server 8000
open http://localhost:8000/dashboard/
```

### "Dashboard shows no data"

**Cause:** Analytics pipeline hasn't been run.

**Solution:**

```bash
# Copy sample data
cp nc-2002-2026-sample.xlsx nc-2002-2026.xlsx

# Run analytics
bash analysis/run.sh

# Refresh dashboard
```

### "Schema validation failed"

**Cause:** Excel file is missing required columns.

**Solution:** Check the error message for missing columns. Either:
1. Update your Excel export to include all required columns
2. Or update column mappings in `analysis/01_ingest.sql`

See `analysis/README.md` for schema drift remediation.

### Charts not rendering

**Cause:** Missing or empty CSV files.

**Solution:**

```bash
# Verify output files exist
ls -la analysis/out/*.csv

# Re-run analytics
bash analysis/run.sh
```

### Tables not sortable

**Cause:** Browser compatibility issue.

**Solution:** Use a modern browser (Chrome, Firefox, Safari, Edge).

## Customization

### Modify Dashboard

Edit `dashboard/index.html`:
- Styling: Tailwind CSS classes
- Charts: Chart.js configuration
- Data loading: Vanilla JS (no framework)

### Add New KPI

1. Add metric to `analysis/03_metrics.sql` or `04_advanced_analysis.sql`
2. Add export in `analysis/run.sh`
3. Update `dashboard/index.html` to read new CSV
4. Add glossary entry in `docs/DASHBOARD_GLOSSARY.md`

### Change Visual Theme

Edit CSS in `dashboard/index.html` `<style>` block:
- Colors: Update `.kpi-accent-*` classes
- Layout: Update `.panel`, `.table-shell` classes
- Typography: Update `.mono`, `.badge` classes
